## Instructions

Place your 5 IEEE papers related to diabetes and cardiovascular disease in this folder.

### Recommended Topics:

- Heart attack (myocardial infarction) diagnosis

### File Naming Suggestions:
- `diabetes_management_guidelines.pdf`
- `type2_diabetes_treatment.pdf`
- `diabetes_cardiovascular_disease.pdf`
- `myocardial_infarction_diagnosis.pdf`

## Finding IEEE Papers

1. Visit: https://ieeexplore.ieee.org/
2. Search for: "diabetes management" or "myocardial infarction" or "cardiovascular disease"
3. Filter by: Open Access (if possible)
4. Download PDFs and place them here
